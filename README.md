# Тесты для управления механической кистью

---

1) test1.c - Поочередное разгибание, а затем поочередное сгибание пальцев. Алгоритм начинает работу с большого пальца и, соответсвенно, им же заканчивает.
2) test2.c - Одновременное разгибание большого, среднего и указательного пальцев. Пауза в 10 секунд и затем идет одновременное сгибание тех же пальцев. Вновь пауза в 10 секунд.
3) test3.c - Используются 3 комбинации: виктори, шака, коза.
